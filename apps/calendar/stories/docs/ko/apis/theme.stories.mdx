# 테마

## 설명

색상, 배경색 등을 변경하여 원하는 테마를 적용할 수 있다. `setTheme` 메서드를 통해 자유롭게 테마를 변경할 수 있다.

```js
const cal = new Calendar('#container', {
  theme: {
    week: {
      today: {
        color: 'blue',
      },
    },
  },
});
cal.setTheme({
  week: {
    today: {
      color: 'red',
    },
  },
});
```

## 테마 객체

테마 객체는 객체는 공통으로 적용되는 `common`, 주간/일간뷰를 위한 `week`, 월간뷰를 위한 `month` 세 부분으로 나뉘어진 중첩 객체다.

```ts
interface themeObject {
  common: CommonTheme;
  week: WeekTheme;
  month: MonthTheme;
}
```

```ts
interface CommonTheme {

  saturday: { color: string };
```
