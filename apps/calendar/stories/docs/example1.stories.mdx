# Creating calendar instance

Here's a simple example of how to create a calendar instance.

<Canvas>
  <CalendarExample
    options={{
      defaultView: 'month',
    }}
    onInit={(cal) => {
      console.log('Calendar initialized', cal);
      cal.setDate('2022-03-04');
    }}
  />
</Canvas>
